# 💹💵 E-Commerce Admin Dashboard <a name="about-project"></a>

**E-Commerce Admin Dashboard** is a simple dashboard for e-commerce admins, it allows them to add products, upload images , change prices, view orders report create users.

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project, you need:

- Visual Studio Code
- Node.js
- Docker

### Setup

Clone this repository to your desired folder:

```sh
cd <desired folder to contain project>
git clone https://github.com/johnpatrick254/E-Commerce-Dashboard
cd <project_folder>
```

### Install

Install project dependencies:

```sh
npm install
```

### Usage

Start development server:

```sh
npm start
```
