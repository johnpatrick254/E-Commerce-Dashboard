# 📖 E-COMMERCE ADMIN DASHBOARD <a name="about-project"></a>

<!-- > Describe your project in 1 or 2 sentences. -->

This project is used to practise MERN/Typescript. The webapp aims to mirror the admin dashboard on an e-commerce site where the admin can add, edit and delete listed products as well as create users that can then manage the products based on assigned roles. The web app also provides charted reports on sales and profit/loss.

## 🛠 Built With <a name="built-with"></a>

- **Node/Typescript**
- **Express**
- **MySQL**
- **TypeORM**
- **React/TypeScript**
- **RTK Query**
- **Sass**
- **Recharts**
- **Chartjs**

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:

- Visual Studio Code
- APPMS or WAMPP
- A Browser
- Git

### Setup

Clone this repository to your desired folder:

Example commands:

```sh
  cd <desired folder to contain project>
  git clone git clone https://github.com/johnpatrick254/E-Commerce-Dashboard.git
  cd E-Commerce-Dashboard
```

### Install

Install this project with:

```sh
npm install
```

### Usage
First add your mysql credentails in the orm.config inside the Backend folder,
then start your mysql server and execute the following command in root folder:

```sh
cd fronted
npm run dev
```
